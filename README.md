## **MaterialRatingBar**
    评论进度的view
## 使用方法

build.gradle文件中添加:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
并且:

```gradle
dependencies {
    compile 'com.github.ashLikun:MaterialRatingBar:{latest version}'


}
```
